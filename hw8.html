Explanation of AJAX

In the HTML file, you will notice some JavaScript code that uses AJAX.
AJAX, short for Asynchronous JavaScript and XML, allows us to send and receive
data from a server without reloading the entire page. It makes web experiences more
dynamic and interactive by fetching data from the server in the background and
updating specific parts of the webpage with the received data.

Now, let's dive into the AJAX calls in the .html file. The code is making requests
to a server to retrieve data in XML format. It does this by creating an XMLHttpRequest
object, which is a commonly used tool for AJAX. The object is configured with the appropriate
URL to send a "GET" request to the server. Then, the code sets up an
event listener to handle the response asynchronously.
